 User Stories:
- You can see whether Free Code Camp is currently streaming on Twitch.tv.
-  You can click the status output and be sent directly to the Free Code Camp's Twitch.tv channel.
-  If a Twitch user is currently streaming, you can see additional details about what they are streaming.
- You can filter the streamers by their names in a search box.
